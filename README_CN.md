# 运行项目
### 1 拉取项目到本地
```shell
git clone https://github.com/apple-liumingyang/mothers_day.git
```
如果网速很慢，则可以使用以下命令
```shell
git clone https://gitee.com/apple-liumingyang/mothers_day.git
```

### 2 创建虚拟环境
```shell
python3 -m venv venv
```

### 3 激活虚拟环境
```shell
source venv/bin/activate
```
### 4 安装项目依赖
```shell
python3 -m pip install -r requirements.txt
```

### 5 启动项目
```shell
python3 main.py
```
